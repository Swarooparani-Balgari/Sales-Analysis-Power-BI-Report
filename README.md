## Sales Analysis Report (Power BI)

### Objective
Analyze sales data to evaluate regional performance, category trends, customer contribution, and overall profitability using Power BI.

### Tool Used
- Power BI

### Analysis Performed
- Region-wise sales, profit, and discount analysis
- Category-wise and year-wise sales trend analysis
- Identification of top customers and high-performing cities
- KPI-based performance summary

### Key Insights
- Certain regions generate high sales but comparatively lower profit margins
- Technology category contributes the highest sales compared to others
- Sales show an increasing trend across years
- A small set of customers contributes significantly to total revenue

### KPIs Tracked
- Total Sales
- Total Profit
- Discount
- Shipping Cost
- Record Count
